# グリッチ風画像 ver[1.0]

## ▼このスクリプトは何？
グリッチがかった画像を生成します

<br>

## ▼導入方法
aviutlのscriptフォルダに入れてください。

<br>

## ▼機能のご紹介
- **先頭位置**と**図形割合**でどの位置からどの位置まで画像を生成するか範囲指定します。その範囲に、**進行度**の値で生成します。
- **ランダム変化**にチェックを入れると、**seed**の値などで図形が変化します。
- カスタムオブジェクト版もよういしてあります。

<br>
