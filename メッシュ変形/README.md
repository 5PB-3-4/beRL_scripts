# メッシュ変形スクリプト ver[1.0]

## ▼このスクリプトは何？
Thin Plate Splineを利用したメッシュ変形を行うスクリプトです。

<br>

## ▼導入方法
- aviutlのscriptフォルダに@berlsフォルダを入れてください。
<br>

## ▼機能のご紹介
16個の制御点を動かし、「表示」させると画像が変形します。
